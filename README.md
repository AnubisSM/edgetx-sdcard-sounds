# EdgeTX - Sound packs

This repository contains the files needed to generate the voice packages used in EdgeTX.

The currently supported languages are:
* Czech
* Deutsch
* English
* Spanish
* French
* Italian
* Portuguese
* Russian

The following languages are not yet supported but are under development:
* Hungarian
* Dutch
* Swedish
* Slovak
